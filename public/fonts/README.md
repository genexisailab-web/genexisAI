# Laura Zarate Font

Place the official `Laura Zarate` font files in this directory, for example:

```
public/fonts/LauraZarate-Variable.woff2
public/fonts/LauraZarate-Regular.woff
```

The site references these assets via `@font-face` definitions in `src/app/globals.css`. Replace the placeholder file names with the actual ones provided by your branding team.
